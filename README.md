# Firka Napló

## A projektről

Firka egy alternatív [megnevezetlen naplóra] egy kliens alkalmazás. Az alkalmazás célja
egy modern, felhasználóbarát felület biztosítása a(z) [megnevezetlen napló] rendszerhez.
